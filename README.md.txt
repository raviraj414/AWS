# AWS S3 File Uploader

This script uploads files to an AWS S3 bucket using `boto3`.

## Steps to Use
1. Install dependencies: `pip install boto3`.
2. Replace the bucket name and file details in the script.
3. Run the script: `python upload_to_s3.py`.
